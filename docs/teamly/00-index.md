# **Создание интеграции**

Для того чтобы создать интеграцию, нажмите на кнопку **Добавить интеграцию** в верхнем правом углу.

Затем укажите:

-   Название интеграции — поле будет выводиться в общем списке интеграций;

-   Ссылку для перенаправления — адрес, откуда будут осуществляться запросы к API;

    **Важно:** Ссылка, указанная при создании интеграции, должна совпадать с той, которая указывается при авторизации в системе

-   Описание — поле будет отображаться на детальной странице самой интеграции.

# **Редактирование интеграции**

Чтобы внести изменения в интеграцию,
перейдите на её детальную страницу и нажмите на кнопку **Редактировать**.

У вас откроется поп-ап, в котором вы сможете изменить название и описание интеграции.



Поле **«ссылка для перенаправления»** не подлежит редактированию

![контекстная картинка](https://app.teamly.ru/api/v1/upload/editor/view?file_id=b943a45f-2f99-4f2a-8530-c6b5e70881de&source_id=dfa9a32d-02c8-4f35-95d9-c98ca2e478c0&source_type=article&bx_token=&account_slug=academy)



# **Выключение интеграции**

Чтобы отключить интеграцию,
перейдите на её детальную страницу и нажмите на кнопку **Выключение** . У вас откроется поп-ап для подтверждения действия.



Секретный ключ будет обновлён после включения интеграции автоматически.

![контекстная картинка](https://app.teamly.ru/api/v1/upload/editor/view?file_id=594688f8-8275-42b2-908e-838d4a7bb131&source_id=dfa9a32d-02c8-4f35-95d9-c98ca2e478c0&source_type=article&bx_token=&account_slug=academy)



# **Удаление интеграции**

Чтобы отключить интеграцию,
перейдите на её детальную страницу и нажмите на кнопку **Удаление** . У вас откроется поп-ап для подтверждения действия.


Удаление происходит безвозвратно — поэтому, если вы хотите убрать интеграцию на время, [отключите](https://academy.teamly.ru/space/5019017b-ad03-4c00-bdc0-0952fc1cac88/article/dfa9a32d-02c8-4f35-95d9-c98ca2e478c0#8de3b0b0-4720-43a1-b1e1-3d9fe852fa1e) её.
